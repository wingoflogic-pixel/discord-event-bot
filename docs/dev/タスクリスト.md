# タスクリスト（discord-event-bot）

開発・配布・運用の残タスクを一元管理する内部ドキュメント。**配布版（public `discord-event-bot`）には含めない**（他の dev ドキュメントと同様に除外する。→ R-5）。

- 最終更新: 2026-06-20
- 対象バージョン: v7.0.1
- 関連: [dev-and-release-flow.md](dev-and-release-flow.md) ／ [distribution-and-environments.html](distribution-and-environments.html) ／ [adr/](adr)

## 凡例

- 状態: 🔲未着手 ／ 🔄進行中 ／ ✅完了 ／ ⏸保留 ／ ❓要調査・要決定
- 担当: **CC**（Claude Code で実施可）／ **あなた**（Cloudflare/Discord/GitHub のダッシュボード等、CC 不可）／ **共同**
- 各タスクは `ID` で相互参照する。依存は「→ 前提タスクID」で示す。

## メンテ方法

1. 着手したら状態を 🔄、終わったら ✅ にする。
2. 新タスクは該当セクションに `ID` を採番して追記（ID は使い回さない）。
3. 完了タスクは消さずに「9. 完了済み（記録）」へ移すか、状態を ✅ にして残す（履歴になる）。
4. 決定が必要な論点は「8. オープンな決定事項」に集約し、決まったら本文へ反映。
