# 変更履歴

このファイルは利用者向けのリリースノートです。fork を「Sync fork」で更新する前にご確認ください。

---

## v8.0.1 - 2026-06-28

**DB 変更なし**。グループ分け画面で counter のズレ・制約違反の誤判定・名前変更失敗・保存後の配置消失を一括解消する重要修正です。Sync fork でそのまま受け取れます。

### バグ修正

- **グループ分け画面の全症状の真因を修正**
  - サーバーが返すグループ情報に `uuid` が欠落していたため、UI 側で全グループが同じ識別子として扱われていました。これが counter ズレ・apart 制約が常に違反表示・グループ名変更で `400` エラー・保存後の配置消失といった連鎖症状の根本原因でした。1 か所修正することで、これまでの v8.0.0 で残っていた症状もすべて解消されます。
- レスポンス形状を固定化する回帰テストを追加し、同種の欠落を CI で検出できるようにしました。

## v8.0.0 - 2026-06-28

**DB 変更あり**（migration 0010〜0013 を追加）。Sync fork → Cloudflare Workers Builds の自動デプロイで本番 D1 にも適用されます。新規環境変数の追加はありません。

### 新機能

- **グループ分け機能を追加**（ADR 0015）
  - 開催回ごとに参加者を任意のグループに割り振り、結果をチャンネルへ投稿できます。
  - **自動配置**（バランス考慮の自動振り分け）と **手動ドラッグ&ドロップ**の両対応。
  - **ペア制約**: 「同じグループに」「別のグループに」を、必須/推奨の強さで指定できます（通知単位で共有）。
  - 制約違反は画面上でハイライト＋アラート表示されます。
- **番号割り当てに 2 モードを追加**（ADR 0018）
  - **早い順**: 出欠回答が早かった順に番号を割り当て
  - **ランダム**: 公平なシャッフルで番号を割り当て
- **番号割り当て / グループ分けを標準機能化**（ADR 0017）
  - 以前は通知設定でオプトインだった機能を、すべての通知で常時利用可能に。
- **通知オーバーホール**
  - 回答締切・送信時刻・送信ログ・ペース配信を一括で扱える UI に刷新。
  - 送信履歴はサーバー（guild）単位に分離。
- **Discord スラッシュコマンドの整理**
  - `/recruit` → `/notify` に改名（選択 UI 付き）。
  - `/help` を「使い方ガイド」として刷新。
- **配布ガイド (setup.html) の改修**
  - 暗色テーマ（Midnight）へ刷新。
  - 「ADMIN_TOKEN を忘れたとき」セクションを新設。
  - 文言を自然な表現に整理。

### 改善

- 管理 UI を Discord 風のデザインシステムへ全面刷新（サーフェス階層、フラット化、2 トーン、サイドメニュー sticky、ServerRail にサーバーアイコン表示など）。
- フォーム系モーダルを「本物の子ページ遷移」化（ADR 0016）。ハッシュルーティング＋ UUID 連動で、URL を直接共有・戻る/進む操作にも対応。
- 履歴テーブルの絞り込みフィルタを各カラム単位に整理し、全体検索は撤去。
- 送信履歴の宛先列を表示名で表示（以前は user_id がそのまま出ていた箇所を解消）。
- 送信量バナーは超過時のみ表示。

### バグ修正

- **グループ分け画面の重大バグを一括修正**（v8.0.0 直前に発見・修正）
  - 未保存状態で「保存」を押すと、グループに割り当てたメンバーが全員「未割り当て」に戻ってしまう不具合を修正。
  - グループ名を変更しようとすると `400 group_uuid and name required` エラーが出る場合がある不具合を修正。
  - 制約を追加・削除すると、未保存のドラッグ&ドロップ配置が消えてしまう不具合を修正。
  - 配置を変えても制約違反のアラートが古いままになる場合がある不具合を修正。
  - ペア制約一覧で、現在の参加者でないユーザーが user_id 文字列のまま表示されていた箇所を、登録済みの表示名にフォールバックするよう修正。
- 履歴テーブルの絞り込み欄で日本語（IME）が打てない不具合を修正。
- ハッシュルータが画面遷移しない場合がある不具合を修正。
- 送信履歴タブが hidden のまま表示されない不具合を修正。
- 配布ガイドの細かな表示崩れ（sticky ヘッダ周辺の余白、サブステップ番号の見た目）を修正。

### 内部変更（利用者影響なし・参考情報）

- DB に UUID シャドウキーを追加（6 テーブル）し、Admin API ルートを UUID 受信に切り替え（ADR 0016）。
- D1 マイグレーション `0010`〜`0013` を追加（通知オーバーホール、グループ分けスキーマ、UUID シャドウキー、番号割り当て/グループ分け常時化）。
- Discord Interactions 検証を `discord-interactions` から Web Crypto API の Ed25519 へ置換。
- `wrangler.jsonc` を全ブランチ共通化（`env.staging` 上書き方式）。`database_id` は配布 base から省略し、Workers Builds 側で自動生成。
